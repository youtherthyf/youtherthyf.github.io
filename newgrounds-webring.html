To add the widget: 

        <div id="newgrounds-webring">
            <script
          src="https://youtherthyf-madehouse.neocities.org/NG-widget/webring.js"></script>
          </div>

By default, the none design is chosen. This can be overriden to use the minimal design. See design in Additional Attributes below.

Additional Attributes

Additional HTML attributes can be added to the script tag. These include:

design - Expecting string. Create the webring with a design. Currently, the two options are none or minimal If the design is non-existent, the none design is used.

background - Expecting string. Set the background using a URL.

backgroundalt - Expecting string. Set the alt attribute for screen readers or if the background is not available for any reason.

Welcome to the newgrounds webring

YO. got a personal site? then i want you

this is the newgrounds webring, a place for peeps to share their personal websites, blogs, or art sites and do collabs

for the uninitiated a webring is a series of websites that link to each other. they were extremely popular back in the 90s and early 2000s and continue to be on the indie-web today. the goal here is to create a webring for newgrounds creators(programmers, artists, anything really.) so we can link to and help each other grow and maybe encourage more people to make their own websites
